# Click Translator — 点击即翻译

> 一款 Windows 桌面实时翻译工具，选中文字后触发翻译，结果悬浮显示在光标旁。

---

## ✨ 功能特点

| 功能 | 说明 |
|------|------|
| 🖱 多种触发方式 | 双击 / Ctrl+点击 / Alt+点击 |
| 🌐 多引擎支持 | Google（免费）/ Bing（免费）/ 百度（需Key）|
| 🗺 多目标语言 | 中文/英文/日语/韩语/法语/德语/西班牙语/俄语 |
| 💬 悬浮结果窗口 | 透明圆角、跟随光标、可拖拽 |
| 📋 一键复制 | 点击"复制"按钮复制翻译结果 |
| ⏱ 自动消失 | 10 秒后自动隐藏 |
| 🔔 系统托盘 | 后台运行，托盘图标快速访问设置 |

---

## 🚀 快速启动

### 方式一：双击启动（推荐）
双击 `启动翻译软件.bat`

### 方式二：命令行
```bash
pip install -r requirements.txt
python translator.py
```

---

## 📖 使用方法

1. 启动软件，弹出设置窗口
2. 选择触发方式（默认：双击）
3. 点击「▶ 开始监听」
4. **在任意应用中选中文字**（鼠标拖选 / Ctrl+A / 三击选段）
5. 按所选触发方式触发（如：双击选中区域）
6. 翻译结果悬浮显示在光标旁！

---

## ⚙ 设置说明

### 触发方式
- **双击鼠标左键**（推荐）：先选中文字，再双击任意位置
- **Ctrl + 点击**：按住 Ctrl 再点击
- **Alt + 点击**：按住 Alt 再点击

### 翻译引擎
- **Google / Bing**：完全免费，无需 API Key，需要网络
- **百度翻译**：需在代码中填入 `appid` 和 `secret`（[申请地址](https://fanyi-api.baidu.com/)）
